# Portfolio Blog (GitHub Pages)

A fancy static blog homepage.

## Local preview

Open `index.html` directly or serve with:

```bash
python3 -m http.server 8080
```

## Deploy

Push to GitHub and enable Pages in repo settings.

For **username.github.io** user site, the repository name must be exactly:

`<your-username>.github.io`

Current repo: `portfolio` (this will publish as a project site unless renamed).
